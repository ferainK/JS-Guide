# __입력방법__
- 제목 : #
- 인덱싱(숫자) : 1. 2. 3. 
- 블릿 (점) : - or *
***
- 줄바꾸기 : 엔터 2번해야함
- 굵은글씨 : __ (__두번__)
- 이텔릭체 : _ (_한번_)  ※ 굵게 + 이텔릭 = 3번 입력
- 취소선 : ~~ (~~두번~~)
- 인용문구 : ` ` (`한번`)
***
- 구분선 : *** or --- 3개 이상 입력
- 인용문 : > 입력
  >Ex) 나랏말싸 듕국에 달아
  >>똥싸구잇네
- 코드블럭 : ```` 언어종류 ````` (3개씩)
```java
```
